# 🚀 RADIO PRO - Guía de Inicio Rápido

## 5 Minutos para Obtener Audio Profesional

---

## ⚡ PASO 1: PREPARAR (1 minuto)

### 1.1 Abre el archivo HTML
```
Doble clic en: audio-processor.html
(Se abre en tu navegador predeterminado)
```

### 1.2 Permite acceso al micrófono
```
Te aparecerá un diálogo del navegador:
├─ Haz clic en "Permitir"
└─ Si no aparece, revisa el ícono de bloqueo en la barra de direcciones
```

### 1.3 Selecciona tu dispositivo
```
Panel Izquierdo → "Audio Input":
├─ Selecciona tu micrófono o entrada
└─ Verás opciones como "Built-in Microphone" o "USB Audio Device"
```

---

## 🎙️ PASO 2: INICIAR (30 segundos)

### 2.1 Enciende el procesador
```
Botón grande en Panel Izquierdo:
├─ Haz clic en "⚪ POWER ON"
├─ Cambiará a "⚫ POWER OFF" en color verde
└─ El LED de estado se volverá verde
```

### 2.2 Prueba tu entrada
```
Habla o canta hacia el micrófono:
├─ Observa el medidor "Input Level"
├─ Debe estar en AMARILLO (óptimo)
├─ Si está en ROJO: baja "Input Gain"
└─ Si está en VERDE: sube "Input Gain"
```

---

## 🎯 PASO 3: APLICAR PRESET (1 minuto)

Haz clic en uno de estos botones según tu caso:

### Para voz clara y radio:
```
Panel Izquierdo → "Radio FM" ✅
├─ Perfecto para podcasts y voz clara
└─ Sonido tipo radio profesional
```

### Para conversación natural:
```
Panel Izquierdo → "Voz Clara" ✅
├─ Máxima inteligibilidad
└─ Ideal para conferencias y entrevistas
```

### Para música potente:
```
Panel Izquierdo → "Música" ✅
├─ Bajos profundos y agudos brillantes
└─ Para música de cualquier género
```

### Para bajos enormes:
```
Panel Izquierdo → "Ultra Bass" ✅
├─ Bajos monstruosos y controlados
└─ Cuidado: Requiere Master Volume bajo
```

---

## 🔊 PASO 4: AJUSTAR VOLUMEN (30 segundos)

### 4.1 Observa el medidor de salida
```
Panel Izquierdo → "Output Level":
├─ Barra visual con escala de colores
├─ Meta: Principalmente AMARILLO (-20 a -10 dB)
└─ Si está ROJO: Reduce "Master Volume"
```

### 4.2 Ajusta el Master Volume
```
Deslizador debajo de "Output Device":
├─ Rango: -40 dB a +6 dB
├─ Típicamente: -6 dB a -10 dB
└─ Más bajo = Más seguro, menos clipping
```

---

## ✨ PASO 5: PERSONALIZACIÓN (2 minutos)

### Para más CLARIDAD de voz:
```
Panel Central → "EQ Paramétrico":
├─ Aumenta "High-Mid" a +2 dB
├─ Aumenta "Mid" a +1 dB
└─ Resultado: Voz más clara y definida
```

### Para más BAJOS:
```
Panel Derecho → "Bass EQ":
├─ Aumenta deslizador a +4 dB
├─ Haz clic en "Bass Boost Pro"
└─ Resultado: Bajos profundos pero controlados
```

### Para SONIDO BRILLANTE:
```
Panel Central → "EQ Paramétrico":
├─ Aumenta "Treble" a +2 dB
├─ Aumenta "High-Mid" a +1 dB
└─ Resultado: Sonido aireado y brillante
```

### Para SONIDO CÁLIDO:
```
Panel Derecho → "EQ Gráfico":
├─ Busca frecuencias de 2-4 kHz
├─ Aumenta ligeramente (+1-2 dB)
└─ Resultado: Sonido vintage y cálido
```

## 🎛️ CONTROLES PRINCIPALES EXPLICADOS

### Input Gain
```
¿Qué es? Volumen de entrada del micrófono
Rango: -24 dB a +24 dB
Cuándo usarlo: 
├─ Si Input Level está bajo: AUMENTA (+3 a +6 dB)
├─ Si Input Level está alto: DISMINUYE (-3 a -6 dB)
└─ Meta: Input Level en AMARILLO (-18 a -15 dB)
```

### Master Volume
```
¿Qué es? Volumen final de salida
Rango: -40 dB a +6 dB
Cuándo usarlo:
├─ Para lograr volumen confortable
├─ Para evitar clipping (rojo)
└─ Meta: Output Level en AMARILLO (-12 a -8 dB)
```

### EQ Paramétrico (5 Bandas)
```
¿Qué es? Ecualizador con 5 frecuencias principales

Sub-Bass (20-60 Hz): Bajos muy profundos
├─ +: Sonido más profundo
└─ -: Sonido más limpio

Bass (60-250 Hz): Bajos controlados
├─ +: Impacto, presencia
└─ -: Sonido más claro

Mid (250-2kHz): Cuerpo, riqueza
├─ +: Más cuerpo, más presencia
└─ -: Sonido más claro, menos nasal

High-Mid (2-6kHz): Presencia, claridad
├─ +: Voz más clara, más presencia
└─ -: Sonido más suave, menos agresivo

Treble (6-16kHz): Aire, brillo
├─ +: Más aire, más brillantez
└─ -: Sonido más suave, menos dureza
```

### Compresor Multibanda
```
¿Qué es? Controla dinámicos (diferencia entre sonido fuerte/débil)

Threshold (Umbral): A qué nivel comienza a comprimir
├─ Más bajo: Comprime más sonido
└─ Más alto: Comprime solo lo muy fuerte

Ratio (Relación): Cuánto comprime
├─ 2:1 = Compresión suave
├─ 4:1 = Compresión media
└─ 8:1 = Limitador (máximo)

Attack (Ataque): Qué tan rápido reacciona
├─ Rápido (1-5 ms): Protección inmediata
└─ Lento (20+ ms): Permite transientes naturales

Release (Liberación): Qué tan rápido vuelve a normal
├─ Rápido (30 ms): Respuesta ágil
└─ Lento (150 ms): Respuesta suave y natural
```

## 🎧 MONITOREO EN TIEMPO REAL

### Espectro (FFT)
```
Ubicación: Panel Central

Qué ves: Visualización de frecuencias
├─ Barras coloridas = energía en esas frecuencias
├─ Línea superpuesta = envelope del sonido
└─ Grid = referencia de frecuencias

Qué buscar:
├─ Distribución equilibrada de izquierda a derecha
├─ Sin picos agresivos (excepto en intención)
└─ Energía en bajos, medios y agudos
```

### Medidores de Nivel
```
Input Level (Panel Izquierdo):
├─ Verde (< -20 dB): Demasiado bajo
├─ Amarillo (-20 a -1 dB): Perfecto ✅
└─ Rojo (> -1 dB): Clipping, baja Input Gain

Output Level (Panel Izquierdo):
├─ Verde (< -20 dB): Demasiado bajo
├─ Amarillo (-12 a -8 dB): Perfecto ✅
└─ Rojo (> -2 dB): Reduce Master Volume
```

### LEDs de Estado
```
Verde = Procesamiento activo ✅
Rojo intermitente = Clipping detectado ⚠️
Apagado = Detenido
```

## 🆘 SOLUCIÓN DE PROBLEMAS RÁPIDA

### Problema: No escucho nada
```
Soluciones:
├─ Verifica que "Power" esté encendido (botón verde)
├─ Verifica "Output Device" sea correcto
├─ Sube "Master Volume" a -5 dB
└─ Revisa tu altavoz/auricular esté encendido
```

### Problema: Sonido muy bajo
```
Soluciones:
├─ Sube "Input Gain" a -6 dB
├─ Sube "Master Volume" a 0 dB
├─ Acerca el micrófono a 10-15 cm
└─ Prueba aumentar "Bass Boost" en Panel Derecho
```

### Problema: Clipping rojo constante
```
Soluciones:
├─ BAJA "Input Gain" a -15 dB inmediatamente
├─ BAJA "Master Volume" a -10 dB
├─ Aleja el micrófono 20+ cm
└─ Habla más bajo o en ángulo
```

### Problema: Sonido distorsionado
```
Soluciones:
├─ Reduce "Master Volume" en 5 dB
├─ Reduce "Input Gain" en 3 dB
├─ Reduce cualquier EQ Boost a máximo +6 dB
└─ Si persiste: recarga la página
```

### Problema: Sonido apagado/sordo
```
Soluciones:
├─ Aumenta "Treble" en EQ Paramétrico (+1 dB)
├─ Aumenta bandas altas en EQ Gráfico
├─ Verifica que no esté el Stereo Width en 0
└─ Aumenta "High-Mid" en EQ Paramétrico
```

□ Escuchas el audio procesado en tus auriculares/altavoz

## ⭐ TIPS DE ORO

```
1. Menos es más
   └─ Cambios pequeños de +1 o +2 dB son suficientes

2. Escucha en descansos
   └─ Cansancio auditivo = mal juicio

3. Compara con referencia
   └─ Ten track profesional similar para comparar

4. Evita clipping a toda costa
   └─ Una instancia de clipping echará a perder todo

5. Trusting your ears > Reading graphs
   └─ Si suena bien, probablemente lo es

6. Documentar tus configuraciones
   └─ Toma notas de qué funciona para futuros proyectos
```
